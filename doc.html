GPT Image Playground 接口文档

本文档由当前项目代码整理。接口运行在同站点 Pages Function 中，前端可直接使用相对路径调用。除七牛直传接口外，所有接口返回 JSON；错误格式通常为 { "error": { "message": "..." } }。

Base URL: 当前站点 origin CORS: GET / POST / DELETE / OPTIONS 图片历史: D1 元信息 + 七牛图片 云存储: 七牛 Kodo
注意：七牛公网域名当前配置为 HTTP。HTTPS 页面内直接展示 HTTP 缩略图会受浏览器 mixed content 策略影响；文件链接本身仍按接口返回值输出。

OpenAI 兼容代理

所有 /v1/* 请求会被 Worker 原样转发到上游。页面主要使用 /v1/responses 和 /v1/images/generations。

ANY /v1/* 代理转发

转发到 X-Upstream-Base、环境变量 UPSTREAM_BASE 或默认 https://api.openai.com。如果上游 Base URL 已经以 /v1 结尾，Worker 会避免重复拼接。

请求头

  • Authorization: Bearer ...，未配置 OPENAI_API_KEY 时必填。
  • X-Upstream-Base 可选，覆盖上游地址。
  • Content-Type 会透传给上游。

常用路径

  • POST /v1/responses
  • POST /v1/images/generations
{
  "model": "gpt-image-1",
  "prompt": "一张产品图",
  "size": "1024x1024"
}

Agnes 图片与视频

用于 /agnes 页面。API Key 可从请求体传入，也可以配置 Worker secret AGNES_API_KEY。

POST /api/agnes/generate 生成图片

代理 Agnes 图片生成。无参考图时传 return_base64: true 给上游；有参考图时通过 extra_body.image 传图并要求 b64_json。

请求字段

  • apiKey 可选，Agnes API Key。
  • baseUrl 可选，默认 https://apihub.agnes-ai.com。
  • model 默认 agnes-image-2.1-flash。
  • prompt 必填。
  • size 默认 1024x768。
  • images 可选，最多 10 个 base64/data URL 字符串。

返回字段

  • data.imageData base64 图片。
  • data.mimeType 图片 MIME。
  • data.format png/jpeg/webp。
  • data.raw Agnes 原始响应。
{
  "model": "agnes-image-2.1-flash",
  "prompt": "一张电影感海报",
  "size": "1024x768",
  "images": []
}
POST /api/agnes/videos 创建视频任务

创建 Agnes 视频任务。单张参考图走 image 字段，多张参考图走 extra_body.image。

请求字段

  • model 默认 agnes-video-v2.0。
  • prompt 必填。
  • size 如 1152x768，会拆为 width/height。
  • num_frames 默认 121，范围 1-441，并会修正为 8n+1。
  • frame_rate 默认 24，范围 1-60。
  • imageUrls 可选，最多 10 个 HTTP(S) URL。
  • mode 为 keyframes 时写入 extra_body.mode。

返回字段

  • data.id / data.task_id 任务 ID。
  • data.video_id 视频 ID。
  • data.status 任务状态。
  • data.video_url 完成后的视频地址。
  • data.raw 原始响应。
{
  "prompt": "镜头缓慢推进",
  "size": "1152x768",
  "num_frames": 121,
  "frame_rate": 24,
  "imageUrls": ["http://xjp.zaoj.top/uploads/example.png"]
}
POST /api/agnes/videos/query 查询视频任务

查询 Agnes 视频任务状态。会尝试 /agnesapi?video_id=...、带 model_name 的查询地址，以及 /v1/videos/:task_id。

{
  "model": "agnes-video-v2.0",
  "video_id": "video_xxx",
  "task_id": "task_xxx"
}

图片历史

index、studio、agnes 画廊使用这组接口。新记录支持 storage_provider: "qiniu"，生成原图直接写入七牛；缩略图 URL 由原图 URL 追加 -pre 得到。D1 只保留元信息、对象 key、展示 URL 和尺寸/大小数据；旧记录仍兼容 D1 base64 图片。

GET /api/images 分页列表

按创建时间倒序返回图片记录。使用游标分页。

查询参数

  • limit 默认 50，最大 100。
  • cursor_created_at 下一页游标时间。
  • cursor_id 下一页游标 ID。
  • api_mode 可选，例如 agnes。

返回字段

  • data[] 图片记录和元信息。
  • has_more 是否还有下一页。
  • next_cursor 下一页游标。
  • thumbnail_url / preview_url 站内图片地址。
POST /api/images 创建历史记录

写入图片元信息。新七牛模式由浏览器先把原始生成图直传到七牛，再把对象 key、公开 URL 和尺寸/大小元数据登记到 D1；列表缩略图使用七牛侧 -pre 派生地址。

元信息字段

  • prompt、model、api_mode
  • size、quality、format
  • streamed、reference_count
  • storage_provider 可选，传 qiniu 时表示图片已写入七牛。
  • id、storage_key、thumbnail_url 七牛直传登记时必填。
  • 七牛 key 规则：原图 images/{id}.{ext}，缩略图地址 images/{id}.{ext}-pre。

图片字段

  • thumbnail.mime_type: image/webp、image/jpeg、image/png。
  • 七牛直传登记不传 thumbnail.data_base64；D1 只保存元信息。
  • thumbnail.bytes、width、height。
  • preview.* 七牛直传登记不需要传；列表图使用七牛 -pre 派生地址。
  • 旧的 D1/base64 兼容路径仍可传 thumbnail.data_base64。
{
  "id": "018f7c8e-xxxx",
  "prompt": "一张图片",
  "model": "gpt-image-1",
  "api_mode": "responses",
  "size": "1024x1024",
  "format": "png",
  "storage_provider": "qiniu",
  "storage_key": "images/018f7c8e-xxxx.png",
  "thumbnail_url": "http://xjp.zaoj.top/images/018f7c8e-xxxx.png",
  "thumbnail": {
    "mime_type": "image/png",
    "bytes": 1800000,
    "width": 1024,
    "height": 1024,
    "original_bytes": 1800000,
    "original_width": 1024,
    "original_height": 1024,
    "compression_ratio": 1,
    "quality": 1
  }
}
GET /api/images/:id 详情

返回单条图片记录和站内图片 URL。

DELETE /api/images/:id 删除记录

删除 images 和对应 image_thumbnails 记录。

GET /api/images/:id/thumbnail 详情图

返回详情图二进制响应，带 Content-Type 和长期缓存头。

GET /api/images/:id/preview 列表图

返回列表预览图二进制响应。七牛模式下优先读取原图 URL 追加 -pre 的派生图；如果七牛返回错误，接口返回 204 No Content 空响应，不回退原图。

七牛云存储

公共云存储组件使用这组接口。允许目录前缀包括 uploads/、uploads/local/、uploads/remote/、agnes/、agnes/images/、agnes/videos/。

POST /api/storage/qiniu/upload-token 获取上传凭证

生成七牛直传 token。前端拿到 token 后直接向 uploadUrl 提交 FormData。

请求字段

  • filename 文件名。
  • mimeType 或 contentType。
  • size 文件大小。
  • prefix 保存目录。
  • key 可选，显式指定对象 key。
  • expires 可选，默认 1800 秒。

返回字段

  • data.token 七牛上传 token。
  • data.key 最终对象 key。
  • data.uploadUrl 七牛上传地址。
  • data.publicUrl 公网访问 URL。
GET /api/storage/qiniu/files 文件列表

列出七牛 bucket 文件。接口会给图片补 preview_url = url + "-pre"，给视频补 cover_url = url + "-cover"。

查询参数

  • prefix 默认 uploads/。
  • limit 默认 50，范围 1-1000。
  • marker 七牛分页游标。

返回字段

  • data[] 文件列表。
  • marker 下一页游标。
  • url 原文件公网 URL。
  • preview_url / cover_url 派生资源 URL。
POST /api/storage/qiniu/fetch-url URL 抓取上传

让七牛从公开 HTTP(S) URL 抓取文件到 bucket。私网地址、localhost、内网 IP 会被拒绝。

{
  "url": "https://example.com/file.mp4",
  "prefix": "uploads/remote/",
  "filename": "file.mp4"
}
DELETE /api/storage/qiniu/files/:encodedKey 删除文件

删除七牛对象。公共组件删除主文件时会额外尝试删除对应 -pre 或 -cover 派生文件。

禁用接口

ANY /api/jobs 和 /api/jobs/* 已禁用

Background jobs 已撤回。当前统一返回 404 和 { "error": { "message": "Background jobs are disabled" } }。
